# MangoGrowth

High-fidelity frontend prototype for the provided MangoGrowth dashboard product shell.

## Stack

- React
- Vite
- TypeScript
- react-router-dom
- Recharts
- Vitest

## Included Screens

- Dashboard
- Services
- Account Growth
- Orders
- Billing
- Plans and Subscription
- Credits purchase modal

## Run locally

```bash
npm install
npm run dev
```

## Verification

```bash
npm run test
npm run build
npm run lint
```

## Notes

- All data is local mock data.
- No backend integration is included.
- No real payment, account execution, or social growth execution logic is implemented.
